# Migración HTML/JS/CSS a Angular

Proyecto: migrar un sitio web estático a una SPA en Angular.

Entregables:
- Proyecto Angular funcionando
- Migración de HTML/CSS/JS a Angular
- Rutas implementadas
- README actualizado
- Pull Request hacia `main`

Criterios:
- Compila sin errores
- Estructura Angular correcta
- JS convertido a TS
- Estilos organizados (SCSS)
